반복문을 이용하여 별모양 출력 (직삼각형)

import java.util.Scanner;

public class Test01 {

  public static void main(String[] args) {
    
    int width = inputInt();
    int line = 0; 
    while (line++ < width) {
      drawLine(line);
      System.out.println();
    }
  }
  
  static int inputInt() {
    Scanner keyScan = new Scanner(System.in);
    System.out.print("밑변 길이? ");
    int width = keyScan.nextInt();
    keyScan.close();
    return width;
  }
  
  static void drawLine(int length) {
    int x = 0;
    while (x++ < length) {
      System.out.print("*");
    }
  }
}

고찰 
- 반복문 while을 사용하여 별모양을 출력 
- 직삼각형은 하나씩 출력하고 줄바꿈을 하여 출력한다

반복문을 사용하여 별모양 출력 (마름모)

package com.eomcs.basic.ex06.assignment;

import java.util.Scanner;

public class Test02 {

  public static void main(String[] args) {
    Scanner keyScan = new Scanner(System.in);
    
    System.out.print("밑변 길이? ");
    int width = keyScan.nextInt();
    
    int line = 0;
    while (line++ < width) {
      int x = 0;
      while (x++ < line) {
        System.out.print("*");
      }
      System.out.println();
    }
    line--;
    
    while (--line > 0) {
      int x = 0;
      while (x++ < line) {
        System.out.print("*");
      }
      System.out.println();
    }
    
    keyScan.close();
  }
}
고찰 
- 마름모는 직삼각형에 아래의 작은 삼각형을 추가하는 것으로 만든다


반복문을 사용하여 별모양 출력 (정삼각형)

package com.eomcs.basic.ex06.assignment;

import java.util.Scanner;

public class Test03 {

  public static void main(String[] args) {
    Scanner keyScan = new Scanner(System.in);
    
    System.out.print("밑변 길이? ");
    int width = keyScan.nextInt();
    
    if (width % 2 == 0)
      width--;
    
    int spaceSize = width >> 1;
    int line = 0; 
    while (line++ < width) {
      if (line % 2 == 0) {
        continue;
      }
      
      int count = 0;
      while (count++ < spaceSize) {
        System.out.print(" ");
      }
      spaceSize--;
      
      int x = 0;
      while (x++ < line) {
        System.out.print("*");
      }
      System.out.println();
    }
    
    keyScan.close();
  }

}

고찰
- 정삼각형은 홀수가 출력되게 만든다 
- 또한 짝수를 입력했을때는 홀수가 출력되도록 if문을 이용하여 홀수만 출력될 수 있도록 만든다 
- 즉. 3개의 반목문과 2개의 조건문을 이용하여 코드를 작성한다
