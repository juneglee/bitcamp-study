소프트웨어 환경 
시스템  SW + 개발 SW

시스템 SW : 클라이언트와 서버 운영에 필요  ex) OS, 서버프로그램, DBMS
개발 SW 
- 요구사항 관리 도구 
	ex) JIRA, IBM DOORS, Trello
- 설계/ 모델링
  - UML(통합 모델링선)을 지원
    - 시스템 분석, 설계, 구현 등 시스템 개발 과정에서 상호간의 의사소통을 위해 개발
  - 개발의 전과정 
	ex) DB Deginer, Arogo, Plant
- 구현
  - 실체로 구현
	ex) 이클립스, Intellij, visual studio, Node.js
- 빌드
  - 작성된 소스의 빌드 및 배포, 라이브러리 관리 
	ex) Ant, Gradle, Maven, jenkins
- 테스트 
  - 요구사항에 적합하게 구현되는지 확인하는 구현 테스트
	ex) Junit, HTTP, Unit, Spring  test
- 형상관리도구 (버전관리)
  - 산출물을 버전별로 관리하여 품질을 향상을 지원
	ex) git , CVS, subversion , Merwnia
